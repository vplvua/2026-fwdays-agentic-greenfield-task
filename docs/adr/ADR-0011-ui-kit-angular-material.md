# ADR-0011 — UI-набір фронтенду: Angular Material

- **Статус:** Accepted
- **Дата:** 2026-07-07
- **Замінює / Замінено:** —
- **Трасування:** TC-UI-01 · NFR-COMPAT-01 · BC-PRIN-01 · В-01

## Контекст і проблема

Питання В-01 (вибір UI-набору) було делеговано агентній розробці з фіксацією
у циклі Е-1 (зріз S-01). Застосунок малий (5 екранів, PRD §6), мобільний-
пріоритет (TC-UI-01), але вже з S-02 потрібні готові віджети: форми, діалоги,
селекти, снекбари. Писати їх вручну — робота, що не вміщується в бюджет
циклів (BC-GOAL-01); набір має бути сумісним із зафіксованою архітектурою
фронтенду (ADR-0009: zoneless, OnPush, standalone, signals).

## Драйвери рішення

- Гарантована сумісність з Angular 21 (zoneless, standalone) без очікування
  сторонніх релізів.
- Мобільний-пріоритет і доступність «з коробки» (TC-UI-01, NFR-COMPAT-01).
- Мінімум залежностей і церемонії (BC-PRIN-01); безкоштовно (BC-COST-01).
- Прості, добре відомі патерни — краще генеруються агентами.

## Розглянуті варіанти

- **A — Angular Material v21 (+ CDK):** першостороння бібліотека команди
  Angular; Material 3-теми; реліз синхронний з Angular.
- **B — PrimeNG:** ширший набір компонентів, але стороння каденція релізів
  (ризик відставання від Angular 21/zoneless) і важча тема-система.
- **C — без бібліотеки (чистий SCSS):** нуль залежностей, але кожен віджет
  (діалог, селект, снекбар) — ручна робота у кожному зрізі.

## Рішення

Обрано **A — Angular Material v21 + CDK**:

- Тема — Material 3 (`mat.theme`), одна кастомна палітра; типографіка
  системна.
- Компоненти імпортуються standalone у presentational-шарі (ADR-0009);
  обгортки над Material не заводимо, поки немає ≥2 споживачів.
- CDK доступний для примітивів (overlay, a11y) без додаткових рішень.

## Наслідки

- **Позитивні:** віджети для S-02…S-07 готові; сумісність з zoneless/OnPush
  гарантована; В-01 закрито, В-04 (design-check у verify) стає актуальним.
- **Ціна:** візуальна «матеріальність» UI (прийнятно для POC); бандл більший
  за чистий SCSS — контролюємо, імпортуючи лише потрібні компоненти.
